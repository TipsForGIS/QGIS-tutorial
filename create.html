How to create a new shapefile and new features?

For this tutorial, we will use a plugin named OpenLayers Plugin. This plugin allows us to add basemaps from Google, Bing, and Open Street Map. We will use the basemap as a reference for our digitization. Let's start with this tutorial through the following steps.


1. Open QGIS and go to the plugin menu → Manage and Install Plugins...




2. Now the plugins window is open, type the word openlayers in the Search box. Then a list of plugins appear below, select the right one according to the following figure. Now click the Install plugin button.




Under the web menu, you should see a new added sub menu for the OpenLayers plugin.




3. Select web → OpenLayers plugin → Google Maps → Google Satellite as illustrated in the following figure. If you wish to select another basemap, feel free to do so.




I have zoomed to Manhattan area in New York City. The center coordiantes and the scale level are shown below. You can practice in a different area.






4. Click on the New Shapefile Layer button.




a new window appears with the title New Shapfile Layer.




5. Click on the button highlighted below for selecting the CRS. CRS stands for coordiante refernce system. We need to change it to the web mercator projection to match the Google's projection. Type in 3857 to filter the CRSs. By the way, most web maps use the web mercator projection including OSM, OpenLayers, Carto, Leaflet, and Bing.




6. Now select the type line. Then create a new field called st_name with the text data type and length of 50. Click the Add to fields list button then click OK.




7. A save prompt window will appear. Save the file in a folder of your choice. Since a Mac is used for this tutorial, you don't need to specify the extension ".shp". If you are using Windows, double check if you need to type in the extension or not.




8. Notice that the streets shapefile is added to the Layers Panel. Now double click on the streets' symbology as highlighted below to change it.




9. Change the color and the width of the line. Red and 1.5 values are selected respectively. Select whatever options you like.




10. Now we are ready to create features. Make sure that the streets layer is on top of the Google Maps layer in the Layers Panel. Click on the Toggle Editing button to activate the editing mode.




11. On the map, click anywhere and then move the mouse cursor. Then click again at a second location. You can continue with that and draw multiple line segments. Here we are just drawing one segment. To save the feature, hit the right mouse button. Let's explain it again, click on the start point → click again at the end point → right click to stop digitizing and to save the feature.




12. After the right click, a small window will appear for the attributes. Fill in the id and st_name.




The following attributes were entered for two different features.






13. Click on the Save Layer Edits button highlighted below.




14. The features are now saved. Right click on the streets layer and select Open Attribute Table.




You should see the attributes of the features you created.




This is the end of this tutorial. If you have any question, please use the contact page for help.
